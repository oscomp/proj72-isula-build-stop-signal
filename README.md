# proj72-isula-build-stop-signal

为iSulad 创建容器命令增加stop-signal选项，并实现对应功能

### 描述 

当前 iSulad create、run命令不支持stop-signal，设置容器停止的信号，需要参赛者为create、run命令添加stop-signal选项，并实现对应功能。
